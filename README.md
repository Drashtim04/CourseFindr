**CourseFindr** is an intelligent course discovery platform that recommends relevant online courses based on user interests, budget, and skill level. The system combines a clean, responsive frontend with a data-driven recommendation pipeline to deliver personalized learning suggestions.

---

## Features

* **Personalized Course Recommendations**
  Matches users with courses based on interest domains such as Web Development and Data Science.

* **Advanced Filtering**
  Filter courses dynamically by price, duration, and difficulty level.

* **Modern Responsive UI**
  Glassmorphism-inspired interface with smooth animations and mobile-friendly layouts.

* **Bookmarking & Wishlist**
  Allows users to save and manage preferred courses using browser local storage.

* **Data Processing Pipeline**
  Python scripts for cleaning, transforming, and exporting course datasets for frontend consumption.
